# Linux-ComfyUI-Launcher
A simple launcher that opens ComfyUI as an app (not a browser tab), and other minor QOL improvments.

### Features:
- You dont need to `cd` or `source` to start ComfyUI
- Makes the ComfyUI server display as "ComfyUI Server" in task managers
- Restarts ComfyUI cleanly if already running
- Doesnt duplicate the window when restarting
- Includes a desktop tray icon 
- Opens ComfyUI as an app instead of opening a tab in your browser

## Installation Requirments
Install this into your Python virtual environment
- setproctitle

Install this to your system

## Setup
1. Clone the repo into the root directory of you ComfyUI install
2. Run the setup.py
3. Launch ComfyUI using the shortcut or with Run.sh
